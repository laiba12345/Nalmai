# Nalmai naming decision

Selected product name: **Nalmai**, chosen by the project owner.

Positioning: Nalmai detects confusion, recommends an evidence-informed teaching move, observes the next check, and improves strategy selection within the live lesson.

No availability or trademark claim is made in this document. A formal trademark
and domain review is required before commercial launch.

The application uses Nalmai identifiers throughout. Pre-rename environment
variables and an existing SQLite filename remain accepted only as migration
aliases so deployed installations do not lose configuration or mastery data.
